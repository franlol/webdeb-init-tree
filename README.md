# WiT - Web-development Initial Tree
Script to create a simple initial tree folders for the web development<br>
<br>
Installation:<br>
git clone https://github.com/franlol/webdeb-init-tree<br>
sudo cp ./webdeb-init-tree/wit /bin/wit<br>
<br>
Usage:<br>
wit folder-name<br>
